# Privacy Policy

**Now Extension for Claude**

Last updated: May 7, 2026

## Summary

This extension does **not** collect, store, or transmit any user data.

## What this extension does

The "Now Extension for Claude" Chrome extension automatically prepends the current date and time to messages you send on Claude.ai. The timestamp is generated locally in your browser using the standard Web API and is included only in the message you send to Claude.

## Data we collect

**None.** This extension does not collect any data from users.

- We do not track usage
- We do not collect analytics
- We do not log messages or user input
- We do not transmit any data to external servers
- We do not use cookies
- We do not have any backend or database

## Data stored locally

The only data stored by this extension is a single boolean preference (`enabled` on/off) saved in your browser's local extension storage (`chrome.storage.local`). This data:

- Stays entirely on your device
- Is never transmitted anywhere
- Is removed if you uninstall the extension

## Permissions

This extension requests the following permissions:

- **`storage`**: To remember your on/off preference
- **Host permission for `https://claude.ai/*`**: To read and modify the message input field on Claude.ai for timestamp injection. The extension does not access any other website.

## Third parties

This extension does not share data with any third party because no data is collected.

## Changes to this policy

If this policy changes, the updated version will be posted at the same URL with a new "Last updated" date.

## Contact

For questions, please open an issue at:  
https://github.com/jyagunii-bot/now-extension-for-claude/issues

# プライバシーポリシー（日本語）

**Now Extension for Claude**

最終更新日: 2026年5月7日

## 要約

この拡張機能は、ユーザーデータを**一切**収集・保存・送信しません。

## この拡張機能の動作

「Now Extension for Claude」は、Claude.aiで送信するメッセージの先頭に、現在の日時を自動で付加するChrome拡張機能です。タイムスタンプはあなたのブラウザ内で標準のWeb APIを使ってローカルで生成され、Claudeに送信するメッセージに含まれるだけです。

## 収集するデータ

**ありません。** この拡張機能はユーザーから一切のデータを収集しません。

- 利用状況の追跡なし
- アナリティクスの収集なし
- メッセージや入力内容のログ取得なし
- 外部サーバーへのデータ送信なし
- Cookieの使用なし
- バックエンドやデータベースなし

## ローカルに保存されるデータ

この拡張機能が保存する唯一のデータは、ON/OFF設定（boolean値）のみで、ブラウザのローカル拡張ストレージ（`chrome.storage.local`）に保存されます。このデータは:

- あなたのデバイス内にのみ保存される
- どこにも送信されない
- 拡張機能をアンインストールすると削除される

## 権限について

この拡張機能は以下の権限をリクエストします:

- **`storage`**: ON/OFF設定を記憶するため
- **`https://claude.ai/*`へのホスト権限**: Claude.aiのメッセージ入力欄を読み取り、タイムスタンプを挿入するため。他のウェブサイトには一切アクセスしません。

## 第三者との共有

データを収集していないため、第三者と共有するデータもありません。

## ポリシーの変更

ポリシーに変更がある場合、同じURLに新しい「最終更新日」を付けて更新版を掲載します。
